Pengantar Generic

  • Apa itu Generic? Fitur untuk membuat kode yang bekerja dengan berbagai tipe data
  • Analog: Kotak penyimpanan yang bisa menampung apa saja, tapi dengan label tipe
  • Manfaat:

i. Type safety (keamanan tipe)

ii. Code reusability (penggunaan ulang)

iii. Compile-time checking

Masalah Tanpa Generic

class Box {
  dynamic data;  // Bisa apa saja
  
  Box(this.data);
}
void main() {
  var box = Box(42);
  int value = box.data;  // Runtime error jika bukan int
  // Tidak aman!
}

Solusi dengan Generic

class Box {
  T data;
  
  Box(this.data);
}
void main() {
  var intBox = Box(42);      // T = int
  var stringBox = Box('Hello');  // T = String
    
  int value = intBox.data;        // Aman, pasti int
  // stringBox.data = 123;        // ERROR compile-time
}

Sintaks Generic Dasar

  • Type parameter: , ,
  • Konvensi

i. T = Type

ii. E = Element (collection)

iii. K = Key

iv. V = Value

Type Inference (Inferensi Tipe)

  • Dart bisa menebak tipe generic dari konteks
T myFunc(T value) {
  return value;
}
  
void main() {
    var intValue = myFunc(10); // Dart menginferensi T sebagai int
    print(intValue.runtimeType);  // Output: int
      var stringValue = myFunc("test"); // Dart menginferensi T sebagai String
      print(stringValue.runtimeType); // Output: String
}

Pembatasan Tipe dan manfaatnya

  • Keamanan Tipe yang Lebih Baik: Memastikan bahwa kode hanya bekerja dengan tipe data yang kompatibel, sehingga mengurangi risiko runtime error
  • Peningkatan Keterbacaan Kode: Membuat kode lebih jelas dan mudah dibaca karena kita dapat melihat tipe data yang diharapkan secara eksplisit.
  • Memungkinkan Penggunaan Method dari Tipe Data Tertentu: Memungkinkan kita untuk memanggil method yang spesifik untuk tipe data yang dibatasi
  • Mencegah Penggunaan Tipe Data yang Tidak Sesuai: Mencegah penggunaan tipe data yang tidak sesuai dengan logika kode.
  • Membantu dalam Desain dan Implementasi: Membantu kita dalam merancang dan mengimplementasikan kelas dan fungsi generic dengan lebih terstruktur dan terdefinisi.

Praktik 1 – Generic Class & Fungsi

  • 1. Buat class Repository dengan Method: save(T item) , T getById(int id) , List getAll()
  • 2. Buat fungsi generic swap(List list, int i, int j)
  • Interface: Kontrak yang harus diimplementasi
  • 3. Buat class Triple dengan 3 field
// Repository
class Repository {
  final List _data = [];

  void save(T item) {
    _data.add(item);
  }

  T? getById(int id) {
    if (id >= 0 && id < _data.length) {
      return _data[id];
    }
    return null;
  }

  List getAll() {
    return _data;
  }
}

// Fungsi Generic swap
void swap(List list, int i, int j) {
  T temp = list[i];
  list[i] = list[j];
  list[j] = temp;
}

// Triple
class Triple {
  A first;
  B second;
  C third;

  Triple(this.first, this.second, this.third);

  @override
  String toString() {
    return '($first, $second, $third)';
  }
}

void main() {
  // Repository
  var repo = Repository();

  repo.save("Andi");
  repo.save("Budi");
  repo.save("Citra");

  print("Data ID 1: ${repo.getById(1)}");
  print("Semua Data: ${repo.getAll()}");

  // Swap
  List nama = ["A", "B", "C"];

  swap(nama, 0, 2);
  print("Setelah swap: $nama");

  // Triple
  var data = Triple(
    "Andi",
    20,
    85.5,
  );

  print("Triple: $data");
}

Studi Kasus – Cache Generic

class Cache {
  final Map _storage = {};

  void set(String key, T value) {
    _storage[key] = value;
  }

  T? get(String key) {
    return _storage[key];
  }

  void remove(String key) {
    _storage.remove(key);
  }
}

class Product {
  String name;
  double price;

  Product(this.name, this.price);

  @override
  String toString() {
    return 'Product(name: $name, price: \$${price.toStringAsFixed(2)})';
  }
}

void main() {
  // Cache untuk String
  var userCache = Cache();
  userCache.set('name', 'Alice');

  print('User Name: ${userCache.get('name')}');

  // Cache untuk Product
  var productCache = Cache();
  productCache.set('p1', Product('Laptop', 1000));

  print('Product: ${productCache.get('p1')}');

  // Menghapus data
  userCache.remove('name');
  print('Setelah dihapus: ${userCache.get('name')}');
}

Ringkasan & Latihan

Ringkasan

  • Generic membuat kode lebih aman (type-safe) dan dapat digunakan kembali (reusable).
  • Generic dapat digunakan pada class, fungsi, dan collection.
  • Type inference memudahkan penggunaan generic tanpa menuliskan tipe secara eksplisit.
  • Constraints digunakan untuk membatasi tipe data yang diperbolehkan.

Latihan 1 - Class Stack<T>


class Stack<T> {
  final List<T> _items = [];

  void push(T item) {
    _items.add(item);
  }

  T? pop() {
    if (_items.isEmpty) return null;
    return _items.removeLast();
  }

  T? peek() {
    if (_items.isEmpty) return null;
    return _items.last;
  }
}

Contoh Penggunaan:


var stack = Stack<int>();

stack.push(10);
stack.push(20);
stack.push(30);

print(stack.peek()); // 30
print(stack.pop());  // 30
print(stack.peek()); // 20

List<T> mergeLists<T>(List<T> a, List<T> b) {

Challenge - Sistem Repository Generic


class Repository<T> {
  final List<T> _data = [];

  void add(T item) {
    _data.add(item);
  }

  List<T> getAll() {
    return _data;
  }

  T? getByIndex(int index) {
    if (index >= 0 && index < _data.length) {
      return _data[index];
    }
    return null;
  }

  void remove(T item) {
    _data.remove(item);
  }
}

Contoh Penggunaan:


var userRepository = Repository<String>();

userRepository.add("Alice");
userRepository.add("Bob");
userRepository.add("Charlie");

print(userRepository.getAll());

print(userRepository.getByIndex(1));

userRepository.remove("Bob");

print(userRepository.getAll());

// Output:
// [Alice, Bob, Charlie]
// Bob
// [Alice, Charlie]
